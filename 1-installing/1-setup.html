When you run the MaxTo installer, MaxTo will be installed automatically for your user account on the machine. There are no questions asked during the installation.

MaxTo will start immediately after you've run the installer. It should only take about a second to install.

MaxTo will be installed into %LocalAppData%\MaxTo, and it will keep MaxTo up to date, checking for updates every time MaxTo is started. You can also check for updates manually through the settings.

When using the installer, MaxTo will automatically update itself. You can disable this setting after installing.
