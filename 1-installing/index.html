Installation options

System requirements

MaxTo has only one requirement; .NET Framework 4.6.1 or higher.

MaxTo provides a range of methods to install. You should choose the method suitable for your use case:

  1. Normal installer — installs MaxTo for your user, and updates automatically
  2. MSI installer — installs MaxTo for the entire machine
  3. ZIP archive — no installer, unzip where you want. Suitable for use as a portable application
