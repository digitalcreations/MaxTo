The ZIP installation package lets you put MaxTo wherever you want, but you no longer get automatic updates.

System requirements

When you use the ZIP archive, you are responsible for having the required dependencies installed. You need to install the .NET Framework 4.7.2 or later.

The archive contains all the files required to run MaxTo. All the files inside are digitally signed, so you can verify that they came from us.

Setting the path

If you want to use the CLI, you'll need to add the installation folder to your path. This is the recommended way to do this (on Windows 10):

  1. Hit Windows+Pause/Break on your keyboard. This opens the System Control Panel window.
  2. In the left column, click Advanced system settings.
  3. On the bottom of the page that opens, click Environment Variables.
  4. In the window that opens, in the top half, find the variable called Path and click Edit...
  5. Click Browse... and select your MaxTo installation folder.
  6. Now OK your way out of all the open windows.
  7. Start a new instance of Command Prompt. The change will not affect prompts that were already open.

MaxTo does not work with side-by-side installs. It will only allow one version to run at a time.
