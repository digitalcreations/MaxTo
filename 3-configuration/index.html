Configuring MaxTo

There are a lot of things in MaxTo that can be configured, so try to not get lost.

  1. Recipes — automate your window management.
  2. Window behavior — how do windows behave when they are moved?
  3. Monitor — how does MaxTo determine which monitor gets its own regions?
  4. Compatibility shims — you can adjust MaxTo's workarounds for problems in other software.
  5. Language — pick your language here.
  6. Updates — automatically update MaxTo when a new version becomes available.
  7. Startup — should MaxTo start with Windows?
  8. Error reporting — reports any crashes to us so we can fix them.
  9. License — who owns your copy?
  10. Configuration location — change where MaxTo stores its configuration.
