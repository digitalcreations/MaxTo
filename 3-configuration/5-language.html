Language

Most of MaxTo's user interface is translated.

You can change MaxTo's user interface language, or leave MaxTo to use the system default (if we have a translation for that language).

Changing the language will not take effect until MaxTo is restarted.

Some parts of MaxTo will not be translated. These parts are:

  • Log output. Logs should be readable by us if you encounter an issue.
  • Command names and arguments. The commands should be runnable no matter what language settings you have.
